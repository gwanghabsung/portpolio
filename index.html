Gist 동기화

GitHub Personal Access Token으로 비공개 Gist에 데이터를 자동 저장합니다. 모든 디바이스에서 동기화돼요. 토큰은 이 브라우저에만 저장됩니다.

토큰 발급 방법 (처음이면 펼쳐보세요)
1. github.com/settings/tokens 접속
2. Generate new token (classic) 클릭
3. Note: Portfolio Journal 등 적당히 입력
4. Expiration: No expiration 권장 (또는 1년)
5. Scopes: gist 체크박스만 선택
6. Generate token 클릭하고 토큰 복사 → 아래 붙여넣기
7. 첫 사용시 "새 Gist 만들기" 버튼 클릭하면 자동으로 Gist 생성
